## Image Guidelines

### Site Images
- Aspect ratio: 4:3 or 16:9
- Minimum resolution: 1200x800px
- Format: JPEG (optimized) or WebP
- Should show the sacred site clearly
- Preferably during good lighting conditions

### Region Images
- Aspect ratio: 16:9 or 21:9
- Minimum resolution: 1920x1080px
- Should be evocative of the region's sacred character
- Can include landscapes, temples, or cultural scenes

## Image Sources

When adding images, please ensure:
1. You have proper rights to use the image
2. Attribution is provided where required
3. Images are optimized for web (compressed)
4. Sacred sites are depicted respectfully
